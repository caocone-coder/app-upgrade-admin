# App 版本升级配置管理后台

这是一个用于管理 App 版本升级配置的后台管理系统。

## 功能特性

### 列表页
- 筛选条件：产品、平台、状态
- 数据展示：产品、平台、目标版本、已升级人数、状态、编辑人、上架时间
- 操作功能：新增、导出、详情、编辑、删除

### 新增/编辑弹窗
- **升级基础信息**：产品、平台、目标版本、版本类型
- **升级策略设置**：升级方式、升级位置、升级人数、升级时间、升级策略、屏蔽对象
- **弹窗内容配置**：升级弹窗标题、说明、按钮名称
- **跳转及冷却策略**：跳转行为、冷却时间

## 在线访问

- 主页：https://caocone-coder.github.io/app-upgrade-admin/
- 管理页面：https://caocone-coder.github.io/app-upgrade-admin/admin.html

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端，可直接部署到静态托管服务
- 响应式设计，支持桌面端访问

## 本地开发

直接在浏览器中打开 `index.html` 或 `admin.html` 即可。

## 部署

已部署到 GitHub Pages，每次推送到 main 分支会自动更新。
